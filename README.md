# 🔮 Income Classifier — Logistic Regression

## 📌 Objective
Classify income levels of individuals based on age and years of experience
using Logistic Regression.

## 📊 Dataset
- **Source:** Kaggle — Multiple Linear Regression Dataset
- **Features:** Age, Experience
- **Target:** Income classification
- **Size:** 20 records

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## ⚙️ How It Works
1. Load dataset with age, experience and income columns
2. Preprocess and split into train/test sets
3. Train a Logistic Regression classifier
4. Predict and evaluate model performance

## 💡 What I Learned
- Difference between Linear and Logistic Regression
- How to handle classification problems with Sklearn
- How to evaluate a classifier's performance

## 🚀 How to Run
1. Open `logisticRegression_MN3.ipynb` in Google Colab
2. Upload `multiple_linear_regression_dataset.csv`
3. Run all cells
